Los patrones utilizados son los siguientes:

Patrón MVC

:

Se utilizó este patrón para separar la parte lógica y visual de la aplicación, en el caso de la vista se refiere a los .html que se encuentran en la carpeta templates, el modelo sería la información que se encuentra almacenada en las bases de datos Redis y MySQL; y el controlador sería el archivo app.py donde se implemente la lógica de la aplicación.

Patrón Singleton

:

Se utilizó este patrón se utilizó para la instancia de la base de datos de tal manera que no debemos hacer varías conexiones la base de datos si no que para realizar operaciones a la base de datos únicamente hacemos un llamado a esa clase.
